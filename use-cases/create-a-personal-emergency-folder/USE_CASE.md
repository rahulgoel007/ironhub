### 1. Title

Create a personal emergency folder

### 2. Example prompt

"Help me create an emergency folder with important documents, contacts, medical info, and instructions."

### 3. What the agent does

Walks you through collecting all the essential documents you'd need in an emergency — ID scans, insurance cards, medical info, wills, financial account details, and utility contacts. It organizes everything into a structured folder, saves key emergency contacts, and generates a clear one-page instruction note so anyone (family member, first responder) can act quickly. The final output is a printable summary PDF you can keep physically and digitally.

### 4. Skills & tools used

- Emergency Folder Builder skill — guides you through what to include in a personal emergency dossier (ID, medical, financial, legal), prompts for missing items, and structures everything by category
- File Organizer tool — stores and names uploaded documents in a consistent, easy-to-navigate folder structure
- Contacts tool — saves emergency contacts (doctor, lawyer, family, insurance) with phone numbers and roles
- Document Export tool — generates a printable one-page emergency summary with key contacts, instructions, and folder index
